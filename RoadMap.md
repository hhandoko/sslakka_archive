## 1.0
The primary goal for v1.0 release is a stable, usable application with the minimum set of features (i.e. MVP).

### Features: ###
 * [ ] User authentication
 * [ ] Channel operations
   * [ ] Create and modify channel
   * [ ] Join a channel
 * [ ] Post message
 
### Task Checklist: ###

 * [ ] Error logging (ELMAH)
 * [ ] Web Application and REST API (ServiceStack)
